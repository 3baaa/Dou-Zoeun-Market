### 💡 :: Commit Type ::

- feat : 새로운 기능 추가
- fix : 버그 수정
- docs : 문서 수정
- refactor : 코드 리팩토링
- test : 테스트 코드, 리팩토링 테스트 코드 추가
- chore : 빌드 수정

---

### 💡 :: Branch명 ::

`Commit Type/이슈명`

ex) feat/Login

### 💡 :: Commit Message ::

`Commit Type/이슈명 : Commit Message`

ex) feat/Login : 로그인 기능 추가
ex) fix/Login : 소셜 연동 실패 원인 제거

---
